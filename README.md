# Портфолио 1С-разработчика

## О проекте
Этот репозиторий содержит мои учебные и практические наработки по платформе 1С:Предприятие.

## Что внутри

### 🗄️ Исходный код (`src/Configuration/`)
- Распакованная выгрузка конфигурации в XML
### 💾 Готовая база (`dist/`)
- `PracticumProject.dt` — дамп информационной базы с данными
- Для быстрого развёртывания проекта

### 📸 Скриншоты (`docs/screenshots/`)
- Отчёты по заказам, зарплате, прибыли
- Оборотно-сальдовая ведомость

## Используемые технологии
- 1С:Предприятие 8.3
- Система компоновки данных (СКД)
- Регистры накопления и бухгалтерии
- Управляемые формы
